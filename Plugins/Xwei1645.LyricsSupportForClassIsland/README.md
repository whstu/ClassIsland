# 岛上歌词 LyricsSupportForClassIsland
岛上歌词是一款适用于ClassIsland的插件，用于以提醒方式显示歌词。

## 使用说明 Instructions
for 0.0.1 在提醒设置中，将“歌词提醒”的提醒音效设置为目标歌曲，并且选择lrc歌词文件，填写“提醒持续时间”（通常为歌曲时长）和“提醒时间”（定时显示提醒的时间）即可。
